Reglas de precios

Si el producto tiene precio especial (priceE), se usa automáticamente.

Si no, se aplica la lista seleccionada (4, 5 o 6).

Notas

Los códigos de producto no distinguen mayúsculas/minúsculas.

Los totales se calculan automáticamente con IVA y tipo de cambio.
